# 类型定义

interface DetailCardProps {
  size?: number;
  title?: string
  type?: string
  rarity?: 1 | 2 | 3 | 4 | 5
  imgUrl?: string
  main?: {
    key: string
    value: string
    [key: string]: any
  }
  sub?: {
    key: string
    value: string
    [key: string]: any
  }
}

# 参数定义

# Card

参数说明类型是否可选默认值
showStars展示星星boolean可选false
star星级1 | 2 | 3 | 4 | 5必须1
imgUrl图片地址string必须-
desc描述string可选-
foucsing聚焦:表现为选中效果boolean可选false

# DetailCard

参数说明类型是否可选默认值
title标题string可选Title
type类型string可选-
rarity稀有度1 | 2 | 3 | 4 | 5可选1
imgUrl图片地址string可选-
main主属性Property可选-
sub副属性Property可选-
size卡片大小number可选50
